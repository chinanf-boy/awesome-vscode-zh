# 皮棉与智力

万一林肯那可怕的涅盘还没有被释放给你:

> lint最初是给一个特定的程序起的名字,这个程序在C语言源代码中标记了一些可疑的和不可移植的构造(可能是bug).这个术语现在普遍应用于标记任何计算机语言编写的软件可疑用法的工具.

与其他一些编辑器不同,VS代码支持智能感知、链接、开箱即用大纲,并且不需要任何单独的扩展来运行linter包.有些LTENS已经集成在VS代码中,您可以在正式文档中找到完整的列表,[语言文字](https://code.visualstudio.com/Docs/languages/overview)部分.
