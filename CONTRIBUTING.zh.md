
# 贡献准则

要提交一个包到这个列表,请做如下:

1.  找到一个链接到你认为是awesome的包.使用[SudialPosio.VisualStudio.com](https://marketplace.visualstudio.com/)的链接,而不是github存储库中的链接.
2.  为你的包找到合适的地方.如果它属于某一部分,把它放在那里.如果它不属于一个特定的类别,你应该把它放在`uncategorized`部分. 务必把你的包按字母表排列.
3.  如果可能的话,找一个显示软件包功能的`.gif`动画.

规范样式,提交您的包:

```gfm
#### [Title of the package](https://marketplace.visualstudio.com/items?itemName=NAME)
> A short description of what this package does and why is it awesome.

![title of the package](link to the .gif file that explains its functionality)
```

**提交主题**

1.  提交[视频主题](https://vscodethemes.com)链接,如果适用,否则使用[marketplace](https://marketplace.visualstudio.com/)链接.
2.  按字母顺序添加主题.
3.  每个出版商只有一个主题(你可以改变主题).
4.  提交类似的样式截图.您可以使用[这个草图文件](./themes/screenshots/awesome-vscode.sketch)(帮助),内容是[这个模板](./themes/theme-template.js)**提示**将VSCode与画布对齐.

不遵守这些有助于指导方针将导致您的提交请求被拒绝.

谢谢你的建议!
